# udemy lec 

- Identifies (or names) in Lua can be any string of letters, digits and underscores
  - Avoid beginning with a digit or starting with an underscore(reserved for anonymous variable)

  ### Reserved Words
  - and 
  - break 
  - do  
  - else  
  - elseif 
   - end 
   - false 
   - for 
   - function  
   - goto  
   - if  
   - in  
   - local 
   - nil 
   - not 
   - or  
   - repeat  
   - return  
   - then  
   - true  
   - until 
   - while

  ---
  -- lua is case-sensative

  "--" basic comment \
  --[[this is my comment]] \
  
  lua needs no separator between consecutive statements \
  a = 1 b = a * 2 ~ is a valid line of code but bad coding style \
